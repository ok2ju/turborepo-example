# Getting started

<Description>A quick tutorial to get you up and running.</Description>

#### Install core library

Install the component from your command line.

```js
npm install @te/core-react
```

#### Import components

Import desired component from library.

1. You can import all components from root of the package:

```js
import { Button } from "@te/core-react";
```

2. Or you can configure your environment to import specifig components by strict path:

```js
import { Button } from "@te/core-react/button";
```

```js
// tsconfig.json
{
  ...
  "module": "ESNEXT",
}
```

Doc: [link](https://github.com/jkrems/proposal-pkg-exports/)
See this [issue](https://stackoverflow.com/questions/70296652/how-can-i-use-exports-in-package-json-today-for-nested-submodules-and-typescri)

#### Apply styling

Choose appropriate styling approach.

```css
// globals.css
import "@te/core/styles.css";
```

or if you are using `tailwind-cli`

```js
// tailwind.config.js

/** @type {import('tailwindcss').Config} */
module.exports = {
  content: [
    ...config.content_local,
    "./pages/**/*.{js,ts,jsx,tsx}",
    "./components/**/*.{js,ts,jsx,tsx}",
  ],
  theme: config.theme,
  plugins: config.plugins,
};
```
