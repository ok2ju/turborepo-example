# Typography

Typography is used to create clear hierarchies, useful organizations, and purposeful alignments that guide users through the product and experience. It is the core structure of any well designed interface.

We use sans serif native system font families so that products load quickly and can be read efficiently across all devices. We uses a clear naming approach and type tokens to manage typography across complex and layered layouts and patterns.
