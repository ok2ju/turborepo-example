# Spacing

<Description>
  Spacing is the negative area between elements and components. It is commonly
  controlled in code with margin and padding.
</Description>

The spacing scale complements the 2x Grid and typography scale by using multiples of two, four, and eight.

### Spacing scale

Use the spacing scale when building individual components. It includes small increments needed to create appropriate spatial relationships for detail-level designs.
